The bridge from 4.7

🔵 Theory: linear regression is a NN with 0 hidden layers. Add layers → non-linear expressiveness.

Network architecture

Each connection = weight. Each node = linear transform + activation. Hidden activations = embeddings.

Forward pass

\[ \mathbf{z}^{(l)} = W^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)} \quad,\quad \mathbf{a}^{(l)} = f(\mathbf{z}^{(l)}) \] 

z1 = X @ W1 + b1;    a1 = relu(z1)
z2 = a1 @ W2 + b2;   a2 = relu(z2)
z3 = a2 @ W3 + b3;   y_hat = z3

Loss function

Regression: Mean Squared Error

\[ \mathcal{L} = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 \]

Classification: Cross-Entropy

\[ \mathcal{L} = -\frac{1}{n} \sum_{i=1}^{n} \sum_{c=1}^{C} y_{ic} \log \hat{y}_{ic} \]

Backpropagation — chain rule

\[ \frac{\partial \mathcal{L}}{\partial W^{(1)}} = \frac{\partial \mathcal{L}}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial \mathbf{a}^{(2)}} \cdot \frac{\partial \mathbf{a}^{(2)}}{\partial \mathbf{z}^{(2)}} \cdot \frac{\partial \mathbf{z}^{(2)}}{\partial \mathbf{a}^{(1)}} \cdot \frac{\partial \mathbf{a}^{(1)}}{\partial \mathbf{z}^{(1)}} \cdot \frac{\partial \mathbf{z}^{(1)}}{\partial W^{(1)}} \]
🟢 Foundations: just the product rule from calculus, applied layer by layer from output to input.

Gradient descent

\[ W \leftarrow W - \eta \, \frac{\partial \mathcal{L}}{\partial W} \]

Activation functions

Dropout — regularisation by deletion

Randomly zero out neurons during training (p = 0.1–0.5). Forces redundant representations. Off at inference.

Batch normalisation

\[ \hat{x}_i = \frac{x_i - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}} \;,\quad y_i = \gamma\,\hat{x}_i + \beta \]
  • Normalise each layer's inputs to zero mean, unit variance.
  • $\gamma$, $\beta$ are learnable — network can undo the normalisation if needed.
  • Stabilises training, enables higher learning rates, slight regularisation effect.

Learning rate schedules + early stopping

Hidden layers = automated feature engineering

🟣 Advanced: hidden activations are learned representations. First layer: simple features. Deeper layers: complex compositions. Output layer: supervised prediction on learned features. This is representation learning.
  • Hidden layers perform unsupervised feature discovery.
  • Output layer performs supervised prediction.
  • Node values = embeddings (same as 4.7, but non-linear).

def forward(X, W1, b1, W2, b2, W3, b3):
    z1 = X @ W1 + b1;   a1 = np.maximum(0, z1)  # ReLU
    z2 = a1 @ W2 + b2;  a2 = np.maximum(0, z2)
    z3 = a2 @ W3 + b3;  y_hat = z3
    return z1, a1, z2, a2, z3, y_hat

Common mistakes

  1. Zero init — symmetry never broken. Use He init for ReLU.
  2. Dropout at inference — predictions become noisy. Turn it off.
  3. LR too high — loss oscillates. Start 10⁻³, use schedule.
  4. No early stopping — overfits for 1000 epochs when 200 was enough.
