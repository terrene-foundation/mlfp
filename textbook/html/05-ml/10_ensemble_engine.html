Chapter 10: EnsembleEngine

Overview

EnsembleEngine combines multiple trained models into stronger composite predictors with four methods: blending (voting), stacking (meta-learner), bagging (bootstrap aggregating), and boosting (AdaBoost). All accept polars DataFrames and auto-detect classification vs regression.

Concepts

Ensemble Methods

MethodHow It WorksWhen to Use
blend()Average predictions or majority voteDiverse model combination
stack()Meta-learner on CV predictionsModels make different errors
bag()Bootstrap sample copies of one modelReduce variance (trees)
boost()Sequential, upweight misclassifiedReduce bias, improve weak learners
Soft vs Hard voting: Soft voting averages probabilities (requires probabilistic models). Hard voting takes majority class vote (works with any classifier).

Key API Reference

MethodPurpose
engine.blend(models, data, target, method, weights, ...)Voting ensemble
engine.stack(models, data, target, meta_model_class, fold, ...)Stacking ensemble
engine.bag(model, data, target, n_estimators, ...)Bagging ensemble
engine.boost(model, data, target, n_estimators, learning_rate, ...)Boosting ensemble

Code Walkthrough

Step 1: Train Base Models

from sklearn.ensemble import RandomForestClassifier, GradientBoostingClassifier
from sklearn.linear_model import LogisticRegression

rf = RandomForestClassifier(n_estimators=20, random_state=42).fit(X, y)
gb = GradientBoostingClassifier(n_estimators=20, random_state=42).fit(X, y)
lr = LogisticRegression(max_iter=200, random_state=42).fit(X, y)

Step 2: Blend

from kailash_ml.engines.ensemble import EnsembleEngine

engine = EnsembleEngine()
blend_result = engine.blend(
    models=[rf, gb, lr], data=df, target="target",
    method="soft", test_size=0.2, seed=42,
)
assert blend_result.n_models == 3
assert "accuracy" in blend_result.metrics

Step 3: Stack

stack_result = engine.stack(
    models=[rf, gb, lr], data=df, target="target",
    meta_model_class="sklearn.linear_model.LogisticRegression",
    fold=3, test_size=0.2, seed=42,
)

Step 4: Bag and Boost

from sklearn.tree import DecisionTreeClassifier
dt = DecisionTreeClassifier(random_state=42).fit(X, y)

bag_result = engine.bag(
    model=dt, data=df, target="target",
    n_estimators=15, max_samples=0.8, test_size=0.2, seed=42,
)

boost_result = engine.boost(
    model=dt, data=df, target="target",
    n_estimators=30, learning_rate=0.1, test_size=0.2, seed=42,
)

Common Mistakes

MistakeWhat HappensFix
Empty model listValueErrorProvide at least one model
Invalid blend methodValueErrorUse "soft" or "hard"
Weight count mismatchValueErrorMatch weights to models count
Non-allowlisted meta modelValueErrorUse sklearn-prefixed classes

Key Takeaways

  • Four methods (blend, stack, bag, boost) cover major ensemble strategies.
  • Bagging reduces variance; boosting reduces bias.
  • Task type is auto-detected from the target column.
  • Component contributions show per-model performance within the ensemble.
