Module 4 Unsupervised Machine Learning and Advanced Techniques for Insights

8 lessons ~30 hours total study Scaffolding: 40% (light+) Audience: completed M1 – M3

The conceptual pivot of the entire MLFP curriculum. Pattern discovery without labels, then the bridge to neural feature learning. You will watch feature engineering slide from human craft (M3) to statistical discovery (4.1 – 4.6) to optimisation-driven embeddings (4.7) to neural representation learning (4.8) — and walk out able to explain why hidden layers are automated feature engineering with error feedback.

Module Outcomes

By the end of this module, you will be able to:

  • Apply clustering algorithms and evaluate cluster quality with three complementary metrics
  • Implement the EM algorithm from scratch and fit Gaussian mixture models
  • Reduce dimensionality with PCA, kernel PCA, t-SNE, and UMAP — and choose between them
  • Detect anomalies with statistical and ML methods, then blend scores using the EnsembleEngine
  • Discover co-occurrence patterns with Apriori and FP-Growth
  • Extract topics from text using TF-IDF, LDA, and BERTopic with coherence evaluation
  • Build recommender systems including matrix factorisation with ALS
  • Explain how hidden layers are automated feature engineering with error feedback
  • Train a neural network from scratch with the complete training toolkit

The Feature Engineering Spectrum — This Module's Spine

Every lesson in M4 sits on a single axis running from manual to automatic feature construction:

  1. M3 — You designed features by hand (lag features, bucketing, ratios) using domain knowledge.
  2. M4.1 – 4.6 — Unsupervised ML discovers features without labels. Clusters become categorical features. Principal components become continuous features. Topics become text features. No error signal is used — the algorithm finds whatever structure exists in the data.
  3. M4.7 — THE PIVOT — Matrix factorisation learns user and item embeddings by minimising reconstruction error. For the first time, optimisation drives feature discovery. This is where unsupervised and supervised meet.
  4. M4.8 — Neural networks generalise the pivot. Hidden layer activations are embeddings, learned by minimising a loss function through backpropagation. Add non-linear activation functions and 2+ hidden layers can represent any function.
  5. M5+ — Specialised architectures (transformers, GNNs) learn domain-specific features using the same principle.

Watch for the recurring spectrum diagram in Lessons 4.7 and 4.8 — it is the single most important diagram in MLFP.
