Module 5 Deep Learning and ML Mastery in Vision, Sequence, and Transfer

8 lessons ~36 hours total study Scaffolding: 30% (light) Prerequisite: M4 (neural networks)

Every major deep learning architecture — one paradigm per lesson, all implemented end-to-end in PyTorch with the Kailash engines (ModelVisualizer, OnnxBridge, InferenceServer, RLTrainer) doing the heavy lifting around each model.

Module Outcomes

By the end of this module, you will be able to:

  • Build and train autoencoders (vanilla, denoising, VAE, convolutional) with torch.nn
  • Implement modern CNNs with residual and squeeze‑excitation blocks, and export them with OnnxBridge
  • Build RNN, LSTM, and GRU networks for sequence regression and classification
  • Derive self‑attention from scratch and assemble a working Transformer encoder
  • Train GANs (DCGAN‑style and WGAN‑GP) and reason about mode collapse and diffusion alternatives
  • Apply graph neural networks (GCN, GAT) to node and graph tasks
  • Fine‑tune pre‑trained vision and NLP backbones and deploy them via InferenceServer
  • Implement a policy‑gradient RL agent on a Gymnasium environment and relate it to PPO and DQN

The Four Paradigms of Deep Learning

Every architecture in this module lives somewhere on a two‑axis map: what kind of structure the data has (grid, sequence, graph, set), and what the model is supposed to do with it (compress, classify, generate, act). The eight lessons trace a path through that map:

Prerequisites

This module assumes M4 (classical ML + the neural‑network bridge in 4.8): tensors, forward pass, backprop, gradient descent, dropout, batch norm, and optimisers. If any of those feel unfamiliar, re‑read Lesson 4.8 before starting 5.1 — every lesson in this module builds on that foundation.
