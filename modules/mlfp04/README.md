# Module 4: Unsupervised Machine Learning and Advanced Techniques for Insights

**Kailash**: kailash-ml (AutoMLEngine, EnsembleEngine, ModelVisualizer, OnnxBridge) | **Scaffolding**: 40%

## Description

Pattern discovery without labels, then the bridge to neural feature learning. USML = automated feature engineering.

## Lessons (8)

1. **Clustering** — K-means, hierarchical (dendrograms, linkage), DBSCAN, HDBSCAN, spectral, evaluation metrics
2. **EM Algorithm and Gaussian Mixture Models** — Soft clustering, E-step/M-step, 20-line implementation, Mixture of Experts
3. **Dimensionality Reduction** — PCA (SVD), kernel PCA, t-SNE, UMAP, manifold learning
4. **Anomaly Detection and Ensembles** — Z-score, IQR, Isolation Forest, LOF, EnsembleEngine
5. **Association Rules and Market Basket Analysis** — Apriori, FP-Growth, support/confidence/lift
6. **NLP — Text to Topics** — TF-IDF, BM25, Word2Vec, LDA, BERTopic, coherence metrics
7. **Recommender Systems and Collaborative Filtering** — Content-based, user/item CF, matrix factorisation (ALS), THE PIVOT
8. **DL Foundations — Neural Networks and Training Toolkit** — Forward pass, backprop, gradient descent, activations, dropout, batch norm, optimisers
