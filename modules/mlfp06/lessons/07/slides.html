PACT D/T/R addressing

from pact import GovernanceEngine, load_org_yaml, Address

loaded = load_org_yaml("finance_org.yaml")
engine = GovernanceEngine(loaded.org_definition)

# Role address grammar: every D or T is followed by exactly one R
addr = Address.parse("D1-R1-T1-R1")
verdict = engine.verify_action(
    role_address="D1-R1-T1-R1",
    action="read_customer_data",
    context={"data_classification": "confidential"},
)
# verdict.allowed, verdict.reason, verdict.level

GovernedSupervisor

from kaizen_agents import GovernedSupervisor

governed = GovernedSupervisor(
    model="gpt-4o-mini",
    budget_usd=5.00,
    tools=["analyse_customer_data"],
    data_clearance="restricted",
)

async def executor(spec, inputs):
    return {"result": "...", "cost": 0.01,
            "prompt_tokens": 100, "completion_tokens": 50}

result = await governed.run(
    objective="Analyse customer data", execute_node=executor,
)
# result.success, result.budget_consumed, result.audit_trail
⚙ Kailash: every step passes through verify_action. Verdict levels: allowed, blocked, warn, audit. Audit trail is hash-chained and tamper-evident via governed.audit.verify_chain().

Governance testing

from pact import RoleEnvelope, MonotonicTighteningError

def test_denied_access_stays_denied():
    verdict = engine.verify_action(
        role_address="D1-R1-T1-R1",   # analyst
        action="model_deployment",
        context={"data_classification": "restricted"},
    )
    assert not verdict.allowed

def test_envelope_only_tightens():
    # Loosening a child envelope raises structurally:
    with pytest.raises(MonotonicTighteningError):
        RoleEnvelope.validate_tightening(
            parent_envelope, loosened_child_envelope)
⚠ Key principle: test what does NOT happen. Governance tests verify denied access stays denied.

Common mistakes

  1. Fail-open — if governance check fails, DENY. Never default to allow.
  2. Testing only happy paths — test denied access, not just allowed access.
  3. No audit trail — compliance needs proof. Log every access decision.
