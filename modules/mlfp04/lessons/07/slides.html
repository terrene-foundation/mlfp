The Feature Engineering Spectrum

Content-based filtering

  • Represent each item as a feature vector (TF-IDF, metadata).
  • User profile = average of items they liked.
  • Recommend: top-k by cosine similarity.
🟢 Foundations: no other users needed. Works for new users with item features. Cannot discover surprises.

Collaborative filtering

The user-item matrix

Matrix factorisation objective

\[ \min_{U, V} \sum_{(u,i) \in \text{obs}} \bigl(r_{ui} - \mathbf{u}_u^T \mathbf{v}_i\bigr)^2 + \lambda\bigl(\|\mathbf{u}_u\|^2 + \|\mathbf{v}_i\|^2\bigr) \]
  • $\mathbf{u}_u$ = user embedding (k-dimensional), $\mathbf{v}_i$ = item embedding.
  • Predicted rating = dot product of embeddings.
  • $\lambda$ prevents overfitting on sparse data.
THE PIVOT: these embeddings did not exist before training. They are features created by optimisation.

ALS from scratch


def als(R, k=20, lam=0.1, n_iter=20):
    m, n = R.shape
    U = np.random.randn(m, k) * 0.01
    V = np.random.randn(n, k) * 0.01
    mask = (R > 0).astype(float)
    for _ in range(n_iter):
        for u in range(m):
            idx = mask[u] > 0
            V_u = V[idx]
            U[u] = np.linalg.solve(
                V_u.T @ V_u + lam * np.eye(k),
                V_u.T @ R[u, idx])
        for i in range(n):
            idx = mask[:, i] > 0
            U_i = U[idx]
            V[i] = np.linalg.solve(
                U_i.T @ U_i + lam * np.eye(k),
                U_i.T @ R[idx, i])
    return U, V

Hybrid recommenders

  • Combine content-based features with collaborative signals.
  • Mitigate cold start: new items have features but no interactions.
  • Typical: neural network with embedding layer (CF) + feature layer (content).

Most production recommenders at Shopee, Grab, and Netflix are hybrids.

Four common mistakes

  1. Training on the full matrix — hold out test interactions.
  2. Ignoring implicit data — clicks are 100x more abundant than ratings.
  3. Too many latent factors — $k = 20$–$50$ typical; higher overfits.
  4. Missing the pivot insight — if you leave thinking this is about recommendations, re-read.
