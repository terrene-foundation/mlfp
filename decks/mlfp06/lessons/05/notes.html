Speaker Notes — Lesson 6.5AI Agents — ReAct, Tool Use, Function Calling

Total time: ~90 minutes  ·  8 slides  ·  ~11 min per slide (heavy demo)

This is the lesson where the LLM stops being a text generator and becomes an autonomous actor. The ReAct loop is simple conceptually but powerful in practice. Spend 40% of time on live demo: building a real agent that explores data and trains a model autonomously. The cost budget discussion is critical for production — every student who ships an agent without budget will learn the hard way.

Slide 1~3 minTitle

AI Agents

"Until now, the LLM generates text and stops. An agent generates text that includes tool calls, executes those tools, reads the results, and reasons again. It is a brain with hands."

→ "The ReAct loop."
Slide 2~8 minReAct

ReAct loop

Walk the diagram. "Thought: I need to understand this dataset. Action: call explore_data with 'hdb_resale'. Observation: 50,000 rows, 8 columns, prices range from 200k to 1.2M. Thought: I should try a regression model. Action: call train_model with 'regression'. Observation: R-squared 0.87. Thought: Good enough. Final answer: regression with R-squared 0.87." Make the loop concrete.

For beginners: "Think of a junior analyst working through a task. They think, do something, check the result, think again. An agent does the same thing but with LLM reasoning and tool execution."
→ "Function calling makes this structured."
Slides 3–5~20 minFunction calling, design, Kaizen code

Function calling, design framework, and live demo

Walk the function schema, then the 4-step design framework, then the Kaizen code. Live demo: "Watch the agent explore the dataset, decide on preprocessing, choose a model, train it, and generate a report — all autonomously. The cost tracker shows we spent $0.43 on this analysis."

→ "Tool design principles."
Slides 6–8~15 minTools, mistakes, recap

Tool design, mistakes, and recap

Walk tool principles (3–5 tools, clear descriptions, error handling). Walk mistakes (#2 is most dangerous: "I have seen agents spend $200 in 10 minutes stuck in a reasoning loop. Cost budgets are not optional.") Preview 6.6: "One agent is useful. Multiple agents coordinating is a team. Supervisor delegates to specialists. MCP exposes tools at scale."

→ End of lesson.
