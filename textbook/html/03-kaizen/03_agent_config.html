Chapter 3: Agent Configuration

Overview

Kaizen provides framework-wide configuration through KaizenConfig and kaizen.configure(), lifecycle management via AgentManager, and environment-based configuration loading.

Code Walkthrough

Step 1: Global Configuration

import kaizen
from kaizen import KaizenConfig, AgentManager

kaizen.configure(
    signature_programming_enabled=True,
    mcp_integration_enabled=False,
    multi_agent_coordination=True,
    transparency_enabled=True,
)

Step 2: AgentManager

manager = AgentManager()
assert isinstance(manager, AgentManager)

Step 3: Environment Loading

env_config = kaizen.load_config_from_env(prefix="KAIZEN_")
assert isinstance(env_config, dict)
Note: Environment variables use the KAIZEN_ prefix, e.g., KAIZEN_SIGNATURE_PROGRAMMING_ENABLED=true.

Step 4: Framework Entry Point

from kaizen import Kaizen, Framework

assert Kaizen is Framework  # Alias

framework = Kaizen()
assert isinstance(framework, Kaizen)

Key Takeaways

  • kaizen.configure() sets framework-wide defaults.
  • AgentManager provides lifecycle management and agent discovery.
  • load_config_from_env("KAIZEN_") reads from environment variables.
  • Kaizen and Framework are the same class.
  • Configure before creating agents — settings do not apply retroactively.
