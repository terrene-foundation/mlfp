Module 1 Data Pipelines and Visualisation Mastery with Python

8 lessons ~24 hours total study Scaffolding: 70% (heavy) Audience: zero Python required

Zero to productive in eight lessons. Every Python concept grounded in a real Singapore data task — no abstract exercises, no toy examples.

Module Outcomes

By the end of this module, you will be able to:

  • Write Python programs with variables, functions, loops, conditionals, and collections
  • Load, filter, join, aggregate, and transform datasets using Polars (Arrow-backed, vectorised)
  • Create interactive visualisations using Plotly via the Kailash ModelVisualizer engine
  • Profile datasets automatically and detect data quality issues with DataExplorer
  • Build end-to-end data pipelines: load → profile → clean → visualise → report
