# Module 1: Machine Learning Data Pipelines and Visualisation Mastery with Python

**Kailash**: kailash-ml (DataExplorer, PreprocessingPipeline, ModelVisualizer) | **Scaffolding**: 70%

## Description

Zero to productive. Learn Python by exploring real Singapore data. Every Python concept grounded in a data task.

## Lessons (8)

1. **Your First Data Exploration** — Variables, data types, f-strings, Polars basics (read_csv, shape, describe)
2. **Filtering and Transforming Data** — Booleans, pl.col(), filter(), select(), sort(), with_columns()
3. **Functions and Aggregation** — def/return, for loops, lists/dicts, group_by/agg
4. **Joins and Multi-Table Data** — if/else, imports, join types (left/inner/outer), dictionary lookups
5. **Window Functions and Trends** — over(), rolling_mean(), shift(), lazy frames
6. **Data Visualisation** — Gestalt principles, chart selection, Plotly via ModelVisualizer
7. **Automated Data Profiling** — DataExplorer (8 alert types), AlertConfig, compare(), try/except
8. **Data Pipelines and End-to-End Project** — Null handling, ETL, REST APIs, PreprocessingPipeline
