# Module 10: Alignment, RL & Governance

**Kailash**: Align (AlignmentPipeline, AdapterRegistry), kailash-ml (RLTrainer), PACT (GovernanceEngine, PactGovernedAgent), Nexus | **Scaffolding**: 20%

## Lecture (3h)

- **10A** Fine-Tuning: LoRA/QLoRA/DoRA, SFT, data practices, adapter lifecycle, AlignmentPipeline
- **10B** Preference Alignment: RLHF pipeline, DPO derivation, GRPO, ORPO, SimPO, KTO, Constitutional AI
- **10C** Reinforcement Learning: MDPs, Bellman equations, Q-learning, DQN, Policy Gradient, PPO, SAC, GAE
- **10D** Model Operations: merging (SLERP, TIES, DARE), quantization, distillation, ONNX export, AdapterRegistry
- **10E** AI Governance: EU AI Act, Singapore AI Verify, PACT D/T/R grammar, operating envelopes, governed agents

## Lab (3h) — 8 Exercises

1. LoRA fine-tuning with AlignmentPipeline (SFT, parameter savings calculation)
2. DPO preference alignment (preference pairs, beta tuning, safety evaluation)
3. RL fundamentals with RLTrainer (MDPs, value iteration, Q-learning, DQN)
4. PPO training for inventory management (clipped objective, GAE, reward shaping)
5. Model merging and export (linear merge, SLERP, AdapterRegistry, OnnxBridge)
6. AI governance with PACT (YAML org definition, GovernanceEngine, D/T/R, operating envelopes)
7. Governed agents with PactGovernedAgent (budget cascade, tool restrictions, audit chain)
8. Capstone: governed ML system (alignment + PACT + Nexus multi-channel deployment)

Singapore domain Q&A (SFT), preference pairs (DPO), inventory demand, company reports
