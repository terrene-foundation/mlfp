Chapter 6: Presets

Overview

Presets provide one-line middleware configuration for common deployment scenarios. Choose from five presets (none, lightweight, standard, saas, enterprise) or use the NexusEngine builder for production deployments.

Concepts

Progressive Presets

PresetMiddlewarePluginsIncludes
none00Nothing
lightweight20CORS + Security Headers
standard50+ CSRF + Rate Limit + Error Handler
saas54+ JWT + RBAC + Tenant + Audit
enterprise56+ SSO + Feature Flags

NexusConfig

NexusConfig centralizes all settings that preset factories need. Secrets are redacted in __repr__() for safety.

Code Walkthrough

Step 1: Explore Presets

from nexus.presets import PRESETS, get_preset

lightweight = get_preset("lightweight")
assert lightweight.name == "lightweight"
assert len(lightweight.middleware_factories) == 2

# Unknown preset raises ValueError
try:
    get_preset("nonexistent")
except ValueError as e:
    assert "Unknown preset" in str(e)

Step 2: NexusConfig

from nexus.presets import NexusConfig

config = NexusConfig(
    cors_origins=["https://app.example.com"],
    jwt_secret="super-secret-key",
    rate_limit=200,
    environment="production",
)

# Secrets are redacted in repr
repr_str = repr(config)
assert "[REDACTED]" in repr_str
assert "super-secret-key" not in repr_str
Security: NexusConfig.__repr__() automatically redacts sensitive fields like jwt_secret, preventing accidental exposure in logs.

Step 3: Preset via Constructor

from nexus import Nexus

app = Nexus(
    preset="lightweight",
    cors_origins=["https://app.example.com"],
    enable_durability=False,
)
assert app._active_preset == "lightweight"

Step 4: NexusEngine Builder

from nexus import NexusEngine, Preset

engine = (
    NexusEngine.builder()
    .preset(Preset.SAAS)
    .bind("0.0.0.0:8080")
    .config(enable_durability=False)
    .build()
)

assert isinstance(engine.nexus, Nexus)
assert engine.bind_addr == "0.0.0.0:8080"

Step 5: EnterpriseMiddlewareConfig

from nexus.engine import EnterpriseMiddlewareConfig

enterprise_config = EnterpriseMiddlewareConfig(
    enable_csrf=True,
    enable_audit=True,
    enable_metrics=True,
    enable_rate_limiting=True,
    rate_limit_rpm=500,
    enable_cors=True,
    cors_origins=["https://api.example.com"],
)

engine2 = (
    NexusEngine.builder()
    .enterprise(enterprise_config)
    .bind("0.0.0.0:9090")
    .config(enable_durability=False)
    .build()
)

Step 6: Register on Engine

from kailash import WorkflowBuilder

wb = WorkflowBuilder()
wb.add_node("PythonCodeNode", "double", {
    "code": "output = n * 2",
    "inputs": {"n": "int"},
    "output_type": "int",
})
engine.register("double", wb.build(name="double"))
assert "double" in engine.nexus._workflows

Common Mistakes

MistakeProblemFix
Using saas without JWT secretJWT middleware fails at runtimeSet jwt_secret in NexusConfig
Misspelling preset nameValueErrorCheck PRESETS.keys()
Logging NexusConfig with secretsSecurity leakUse repr() which redacts

Key Takeaways

  • Five presets provide progressive middleware stacks.
  • NexusConfig centralizes settings with secret redaction.
  • Nexus(preset="lightweight") is the simplest setup.
  • NexusEngine.builder() provides a fluent builder for production.
  • The Preset enum maintains cross-SDK parity with Rust.
