Nexus: three channels, one codebase

from nexus import Nexus

app = Nexus("ml-platform")
app.add_auth(provider="jwt", roles=["admin", "analyst", "viewer"])
app.add_middleware("rate_limit", requests_per_minute=60)
app.register_service("predict", predict_service)
app.serve(port=8000)  # API + CLI + MCP simultaneously

RBAC authentication

RoleCan do
adminDeploy, configure, full access
analystQuery, explore, limited training
viewerRead-only predictions

Unauthenticated requests → 401. Unauthorised role → 403. Always.

DriftMonitor integration

from kailash_ml import DriftMonitor

monitor = DriftMonitor(
    reference_data=training_df,
    features=feature_columns,
    alert_threshold=0.05
)

# In production: check each batch
drift_report = monitor.check(new_batch)
if drift_report.has_drift:
    alert_team(drift_report.details)

End-to-end verification


# API channel
resp = requests.post(url, headers=auth, json=query)
assert resp.status_code == 200

# Auth: unauthenticated → rejected
bad = requests.post(url, json=query)
assert bad.status_code == 401

# Governance: restricted resource → denied
restricted = requests.post(url + "/admin",
    headers=analyst_auth, json=query)
assert restricted.status_code == 403
