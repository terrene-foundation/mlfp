The language of uncertainty is probability. The operator that updates beliefs is Bayes' theorem. Today we meet both.

Probability is a weight on an event

For any event $A$ in a sample space:

\[ 0 \;\le\; P(A) \;\le\; 1, \qquad P(A) + P(A') \;=\; 1 \]

Joint and conditional probability:

\[ P(A, B) \;=\; P(A \mid B)\,P(B) \;=\; P(B \mid A)\,P(A) \]

Conditional $P(A \mid B) = P(A,B)/P(B)$ just renormalises inside the world where $B$ happened.

Deriving Bayes' theorem

Start from the two equivalent forms of the joint:

\[ P(A \mid B)\,P(B) \;=\; P(B \mid A)\,P(A) \]

Divide both sides by $P(B)$:

\[ P(A \mid B) \;=\; \frac{P(B \mid A)\,P(A)}{P(B)} \]

Posterior = likelihood × prior ÷ evidence.

The COVID ART tree

You test positive — how worried?

Prior: $P(T) = 0.01$. Sensitivity 90%, false-positive 2%.

\[ P(+) = (0.90)(0.01) + (0.02)(0.99) = 0.0288 \] \[ P(T \mid +) = \frac{(0.90)(0.01)}{0.0288} \approx 0.3125 \]
31% — not 90%. Two out of three positive tests are false positives. The base rate dominates when prevalence is low.

Distributions — the shapes of uncertainty

Expected value and variance

\[ E[X] \;=\; \sum_i p_i\,x_i \]

The two forms of variance are equivalent:

\[ \text{Var}(X) = E[(X-\mu)^2] = E[X^2] - \mu^2 \] 

# Expand the square
# E[(X - μ)²] = E[X² - 2μX + μ²]
#             = E[X²] - 2μE[X] + μ²
#             = E[X²] - 2μ² + μ²
#             = E[X²] - μ²

Conjugate priors — Bayes without integrals

Normal–Normal (for a mean, known variance):

\[ \mu_{\text{post}} = \frac{\sigma^2\,\mu_0 + n\,\sigma_0^2\,\bar{x}}{\sigma^2 + n\,\sigma_0^2} \]

Beta–Binomial (for a proportion):

\[ p \mid \text{data} \;\sim\; \text{Beta}(\alpha + k,\, \beta + n - k) \]

The posterior stays in the same family — the update becomes arithmetic on the parameters.

HDB 4-room Tampines — the update


mu_0, sigma_0 = 480_000, 40_000     # prior (2019 baseline)
x_bar, sigma, n = 542_000, 60_000, 30  # recent data

pp = 1 / sigma_0**2
pd = n / sigma**2

mu_post  = (pp*mu_0 + pd*x_bar) / (pp + pd)
sigma_p  = (1 / (pp + pd))**0.5

# mu_post  ≈ 533,000
# sigma_p  ≈   9,600

The posterior sits between prior and sample, pulled toward the data because $n$ gives it more precision.

As $n$ grows, the data wins

Credible vs confidence

Credible interval (Bayesian): "given the prior and the data, there is a 95% probability the true mean is in [a, b]". What most people intuitively want.
Confidence interval (frequentist): "if we repeated the experiment many times, 95% of the computed intervals would contain the true mean". NOT a probability statement about this specific interval.

Lesson 2.2 develops the frequentist CI in full.

The friendship paradox

Your friends, on average, have more friends than you do. Not an insult — a sampling effect.

\[ E[Y] \;=\; \frac{E[X^2]}{E[X]} \;\ge\; E[X] \]

Sampling through relationships over-represents high-degree nodes. The way you sample shapes what you see.

🟢 Practical: surveys through existing users, customer feedback forms, and social-media polls all suffer from this.

Three rookie mistakes

1. Confusing $P(A|B)$ with $P(B|A)$. Sensitivity ≠ positive predictive value. Use Bayes.
2. Ignoring the base rate. "90% accurate" is useless without $P(T)$.
3. Treating priors as fixed. Two analysts with different priors get different posteriors. Data will eventually reconcile them.
