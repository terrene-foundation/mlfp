Module 3 Supervised Machine Learning — Building and Deploying Models

8 lessons ~30 hours total study Scaffolding: 50% (moderate) Audience: MLFP02 complete

Module 3 is the machine-learning engine room. Module 2 gave you statistics and a single linear model. In the next eight lessons you will engineer features from raw data, master the complete supervised model zoo (SVM, KNN, Naive Bayes, trees, random forests, gradient boosting), calibrate and fair-audit every model you train, and deploy a production pipeline with drift monitoring and a Mitchell model card. The running datasets are Singapore credit scoring, HDB resale prices, and the ICU cohort from Module 2.

Module Outcomes

By the end of this module, you will be able to:

  • Engineer domain-driven features and select the most predictive subset with filter, wrapper, and embedded methods
  • Derive the bias-variance decomposition and apply L1, L2, and ElasticNet regularisation with a Bayesian justification
  • Train, tune, and compare the full supervised model zoo: linear, SVM, KNN, Naive Bayes, decision trees, random forests, and gradient boosting (XGBoost, LightGBM, CatBoost)
  • Handle class imbalance with cost-sensitive learning, focal loss, and SMOTE; calibrate probabilities with Platt scaling and isotonic regression
  • Interpret models with SHAP and LIME and audit fairness under the impossibility theorem
  • Orchestrate the end-to-end pipeline with WorkflowBuilder, HyperparameterSearch, ModelRegistry, and DataFlow
  • Deploy a production model with drift monitoring (PSI, KS), a model card, and conformal prediction intervals

Forward Link: Where This Leads

Module 3 closes out the supervised-learning core. Module 4 picks up where 3.1 left off — generalising "feature engineering" to the full feature engineering spectrum: clustering as automated feature discovery ($n \to 1$), dimensionality reduction as compression ($n \to k$), and representation learning as the deep-learning bridge ($n \to m$). Every model you trained in M3 remains usable in M4 — but now the features are learned rather than hand-crafted.
