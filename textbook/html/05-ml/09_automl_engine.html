Chapter 9: AutoMLEngine

Overview

AutoMLEngine automates the full model selection process: quick-train multiple model families, rank by metric, then run deep hyperparameter search on the top candidate. Optional AI agent augmentation follows a "double opt-in" pattern for safety.

Concepts

Three-Phase Process

  1. Quick-train — Train multiple model families with default hyperparameters.
  2. Deep search — HyperparameterSearch on the top-ranked candidate.
  3. Agent augmentation (optional) — AI agents suggest additional strategies.

Agent Double Opt-In

Two opt-ins required: (1) Install pip install kailash-ml[agents] and (2) set agent=True in config. Without both, agents are not used. This prevents accidental LLM costs.

LLM Cost Tracking

LLMCostTracker tracks cumulative API costs. If spending exceeds max_llm_cost_usd, a LLMBudgetExceededError halts agent calls while allowing the algorithmic pipeline to continue.

Code Walkthrough

Step 1: Configure and Run

from kailash_ml.engines.automl_engine import AutoMLEngine, AutoMLConfig

config = AutoMLConfig(
    task_type="classification",
    metric_to_optimize="accuracy",
    direction="maximize",
    search_strategy="random",
    search_n_trials=2,
    agent=False,
)

result = await engine.run(
    data=df, schema=schema,
    config=config, eval_spec=eval_spec,
    experiment_name="automl_classification",
)

assert result.best_model.rank == 1
assert result.baseline_recommendation is not None
assert result.agent_recommendation is None  # agent=False

Step 2: LLM Cost Tracking

from kailash_ml.engines.automl_engine import LLMCostTracker, LLMBudgetExceededError

tracker = LLMCostTracker(max_budget_usd=0.10)
tracker.record("test-model", input_tokens=100, output_tokens=50)

try:
    tracker.record("test-model", input_tokens=100000, output_tokens=100000)
except LLMBudgetExceededError:
    pass  # Budget exceeded

Step 3: Agent Config

agent_config = AutoMLConfig(
    task_type="classification",
    agent=True,           # Opt-in 1
    auto_approve=False,   # Human approval gate
    max_llm_cost_usd=5.0, # Cost cap
)

Key Takeaways

  • AutoMLEngine automates model family selection and hyperparameter optimization.
  • Agent features require double opt-in for safety.
  • LLMCostTracker enforces budget limits on agent API calls.
  • Financial fields reject NaN, Inf, and negative values.
  • Baseline recommendation is always available; agent recommendation only when enabled.
