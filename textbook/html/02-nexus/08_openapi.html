Chapter 8: OpenAPI

Overview

Nexus auto-generates OpenAPI 3.0.3 specifications from registered handlers and workflows. The OpenApiGenerator derives request schemas from function type annotations, maps Python types to OpenAPI types, and produces a complete spec with paths, schemas, and server information.

Concepts

Python-to-OpenAPI Type Mapping

Python TypeOpenAPI Type
strstring
intinteger
floatnumber
boolboolean
listarray
dictobject
bytesstring (format: binary)

Handler vs Workflow Paths

Handlers create POST /workflows/{name}/execute. Workflows additionally create GET /workflows/{name}/workflow/info.

Code Walkthrough

Step 1: Configure Info

from nexus.openapi import OpenApiGenerator, OpenApiInfo

info = OpenApiInfo(
    title="My ML Platform API",
    version="2.0.0",
    description="Production ML inference endpoints",
    contact_name="Platform Team",
    contact_email="platform@example.com",
    license_name="Apache-2.0",
)

gen = OpenApiGenerator(
    info=info,
    servers=[
        {"url": "https://api.example.com", "description": "Production"},
        {"url": "http://localhost:8000", "description": "Development"},
    ],
)

Step 2: Register a Handler

async def predict(model_name: str, features: list, threshold: float = 0.5) -> dict:
    """Run inference on a trained model."""
    return {"prediction": 1, "confidence": 0.95}

gen.add_handler(
    name="predict",
    handler_func=predict,
    description="Run model inference",
    tags=["inference"],
)

The generator inspects predict's annotations: model_name and features become required; threshold becomes optional with default 0.5.

Step 3: Generate Spec

spec = gen.generate()

assert spec["openapi"] == "3.0.3"
assert "/workflows/predict/execute" in spec["paths"]

# Verify schema
schemas = spec["components"]["schemas"]
predict_schema = schemas["predict_input"]
assert predict_schema["properties"]["threshold"]["default"] == 0.5
assert "model_name" in predict_schema["required"]
assert "threshold" not in predict_schema["required"]

Step 4: JSON Output

import json

json_str = gen.generate_json(indent=2)
parsed = json.loads(json_str)
assert parsed["openapi"] == "3.0.3"
Shorthand: OpenApiGenerator(title="Simple API", version="0.1.0") works without creating an OpenApiInfo object separately.

Common Mistakes

MistakeProblemFix
Missing type annotationsParameters default to stringAlways annotate parameters
Forgetting to register handlersEmpty paths in specRegister all handlers first
Complex nested typesMapped to string fallbackUse simple types

Key Takeaways

  • OpenApiGenerator builds OpenAPI 3.0.3 specs automatically.
  • add_handler() derives schemas from Python function signatures.
  • Handlers create POST execute paths; workflows also get GET info paths.
  • generate() returns a dict; generate_json() returns formatted JSON.
  • Python types map directly to OpenAPI types.
