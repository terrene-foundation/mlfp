MLFP — Module 2 Course Home

Module 2 Statistical Mastery for Machine Learning and AI

8 lessons ~28 hours total study Scaffolding: 60% (moderate+) Prerequisite: Module 1

Probability, parameter estimation, hypothesis testing, regression, and causal inference — derived from first principles and applied to real Singapore data. The language of uncertainty that every ML system speaks.

Module Outcomes

By the end of this module, you will be able to:

  • Reason about probability, conditional probability, and Bayes' theorem
  • Estimate parameters using MLE, MAP, and bootstrap methods — and explain when each fails
  • Design and analyse A/B tests with proper power analysis, SRM detection, and multiple-testing corrections
  • Derive and interpret linear and logistic regression from the matrix form upward
  • Apply CUPED and difference-in-differences to analyse experiments beyond naive averages
  • Execute an end-to-end statistical analysis using FeatureStore and ExperimentTracker
