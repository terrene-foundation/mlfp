Module 6 Machine Learning with Language Models and Agentic Workflows

8 lessons ~28 hours total study Scaffolding: 20% (minimal) — capstone 40% Audience: M1–M5 complete

The final module. You have built classical models, deep networks, and transformers. Now you close the loop: prompt LLMs, fine-tune them with LoRA, align them with DPO, ground them in your own data with RAG, wrap them in agents, coordinate those agents, govern them with PACT, and deploy the whole stack through Nexus. By the end of Lesson 6.8 you will have shipped a complete governed AI system.

Module Outcomes

By the end of this module you will be able to:

  • Apply zero-shot, few-shot, and chain-of-thought prompting with Kaizen Delegate and Signature-typed output
  • Fine-tune a base model with LoRA (low-rank adapters) and the kailash-align AlignmentPipeline
  • Align a model to human preferences with DPO and explain the derivation from the RLHF objective
  • Build a production RAG pipeline with chunking, dense + sparse retrieval, re-ranking and RAGAS evaluation
  • Build Kaizen ReActAgents with custom tools, cost budgets, and function-calling schemas
  • Orchestrate multi-agent systems (supervisor-worker, router, A2A) and expose tools via an MCP server
  • Engineer governance with PACT D/T/R addressing, operating envelopes, budget cascading, and audit trails
  • Deploy a governed agent over API + CLI + MCP simultaneously via Nexus and verify end-to-end enforcement
