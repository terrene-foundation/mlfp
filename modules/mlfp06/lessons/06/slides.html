from kaizen_agents import BaseAgent, Delegate

data_agent = BaseAgent(name="DataScientist", ...)
feature_agent = BaseAgent(name="FeatureEngineer", ...)
model_agent = BaseAgent(name="ModelSelector", ...)
report_agent = BaseAgent(name="ReportWriter", ...)

# Supervisor delegates to workers with individual budgets
supervisor = BaseAgent(
    name="Supervisor",
    workers=[data_agent, feature_agent,
             model_agent, report_agent],
    max_llm_cost_usd=5.00)
result = await supervisor.run("Build model for HDB prices")

MCP: a universal protocol for LLM-tool integration

A2A protocol

  • Structured communication between agents.
  • Each message has: sender, recipient, content schema, metadata.
  • Enables agents built in different frameworks to interoperate.

Security considerations

  • Data isolation: agents with different access levels must not leak data.
  • Structured output: use schemas between agents, not free-form text.
  • Audit trail: log every inter-agent message for compliance.

Common mistakes

  1. Too many agents — start with 2–3.
  2. Free-form inter-agent communication — use structured schemas.
  3. No security boundaries — prevent cross-agent data leakage.
